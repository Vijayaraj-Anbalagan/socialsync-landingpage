## üèóÔ∏è HTML5 Semantic Structure

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="John Doe - Web Developer Portfolio">
    <title>John Doe | Web Developer</title>
</head>
<body>
    <header>
        <nav>
            <ul>
                <li><a href="#about">About</a></li>
                <li><a href="#projects">Projects</a></li>
                <li><a href="#contact">Contact</a></li>
            </ul>
        </nav>
    </header>

    <main>
        <section id="hero">
            <h1>John Doe</h1>
            <p>Web Developer & Creative Problem Solver</p>
        </section>

        <section id="about">
            <h2>About Me</h2>
            <article>
                <p>I'm a passionate web developer...</p>
            </article>
        </section>

        <section id="projects">
            <h2>My Projects</h2>
            <article class="project">
                <h3>Project 1</h3>
                <p>Description of project...</p>
            </article>
        </section>
    </main>

    <aside>
        <h3>Latest Blog Posts</h3>
        <!-- Blog posts sidebar -->
    </aside>

    <footer>
        <p>&copy; 2025 John Doe. All rights reserved.</p>
    </footer>
</body>
</html>
```

## üîÑ Git and GitHub Workflow

### Basic Git Commands
```bash
# Initialize a repository
git init

# Add files to staging
git add .

# Commit changes
git commit -m "Initial commit"

# Connect to GitHub
git remote add origin https://github.com/username/repository.git

# Push to GitHub
git push -u origin main
```

### GitHub Workflow
1. **Create Repository** on GitHub
2. **Clone** to your local machine
3. **Make Changes** to your files
4. **Add & Commit** changes
5. **Push** to GitHub
6. **Deploy** with GitHub Pages

## üåê Accessibility Best Practices

### Essential HTML Attributes
```html
<!-- Alt text for images -->
<img src="profile.jpg" alt="John Doe smiling in a professional headshot">

<!-- Semantic headings hierarchy -->
<h1>Main Page Title</h1>
<h2>Section Heading</h2>
<h3>Subsection Heading</h3>

<!-- Form labels -->
<label for="email">Email Address:</label>
<input type="email" id="email" name="email" required>

<!-- Link descriptions -->
<a href="resume.pdf" target="_blank" aria-label="Download resume as PDF">
    Download Resume
</a>

<!-- Language specification -->
<html lang="en">
```
